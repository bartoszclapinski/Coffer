- `--:--` decision: **forecast = fixed + variable, per category, for `anchor + 1 month`** (anchor = latest transaction's month, dashboard convention). Fixed = active outflow flows attributed by `MatchCategoryId` that land in the target month (monthly always; interval > 1 by cadence phase), summed by `TypicalAmount`; null category → an unattributed-fixed line. Suggested limit = total rounded **up to the nearest 10 zł** for headroom. Explicitly a *simple* projection (flat averages + known recurring charges) — no seasonality/trend, labelled as such; accrual offset ignored (forecast by payment month, matching how budgets count spend).
