UI Mockups

Visual reference for implementation. Each mockup is a working HTML file with light + dark theme. The Apple-inspired design system is in shared/design-tokens.css.

How to use these mockups. They are not binding pixel-perfect specs — they show the intended visual language, density, hierarchy, and component behavior. When implementing in Avalonia or MAUI, match the spirit (spacing rhythm, color usage, typography hierarchy) rather than reproducing exact pixel values. The design system tokens in shared/design-tokens.css map directly to Avalonia ResourceDictionary entries and MAUI ResourceDictionary values.
